# Install
- Macos : curl --proto '=https' --tlsv1.3 https://sh.rustup.rs -sSf | sh
- Update Rust Compiler : rustup update

# Helpful Commands
- Get version : 
    - rustc --version
    - cargo --version
- Compile : rustc main.rs
- Cargo : 
    - Create : cargo new hello_cargo
    - Build only : cargo build
    - Buiold & Run : cargo run
